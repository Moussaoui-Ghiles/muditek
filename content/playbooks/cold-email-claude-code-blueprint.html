Why the list sits at 10K to 50K domains

The list size is a money decision.

  • Below 10K: not enough volume for intent, job-change, and funding triggers to compound into pipeline.
  • Above 50K: you burn enrichment credits chasing accounts you will never reach this year.

Inside that band you lock the personas, titles, segments, and brand voice once. That is why the prompt at the end is fast and cheap.

The three skills that run it

Skills are folders of instructions the agent loads when a task matches. Three do the heavy lifting.

Copywriting

The rules it enforces, pulled from thousands of campaigns:

  • Every email is 60 to 80 words.
  • Open give-first, never with credentials.
  • No meeting ask in the first message. The CTA is curiosity, not calendar.
  • Mirror the prospect's language, not the client's.
  • Reference real signals only. When they don't exist, don't invent them.

List Building

Queries the API for people matching the in-scope titles at the target accounts. Dedupes against live campaigns. Verifies. Scrubs the block list. Returns a clean list with every variable populated.

Campaign Assembly

Creates the campaign, uploads the leads, configures the sequence steps and delays, sets up the A/B test on the subject lines, and returns a draft URL for review.

You can build all three over a weekend. Anthropic ships starter templates. Steal a structure and add your judgment.

What the 20 minutes does

Most of it is the API answering, not the agent thinking:

  1. Reads the PDF, pulls the event date, location, and audience.
  2. Opens the ICP file and pulls the in-scope titles.
  3. Opens the ABM list and finds the target accounts.
  4. Calls the enrichment API, capping spend at 2,000 credits.
  5. The API returns 105 verified prospects.
  6. Dedupes against live campaigns. Removes 50.
  7. Runs the block list. 9 more drop.
  8. Final list: 28 prospects, every variable filled.
  9. Drafts the email and two subject lines for the A/B test.
  10. Builds the campaign and hands back a confirmation URL.

The two-minute audit

The agent misses a few things, and the misses are always the same four:

  • Subject lines. Date-led lines win on time-sensitive sends. Push the date to the front.
  • The CTA. The agent drops the link in the body, which depresses replies. Swap it for a soft ask: "want me to send the invite?"
  • The sequence. One step is thin. Add follow-ups at day 4, 9, and 16.
  • The variables. Spot-check that company and first name filled correctly.
The audit is ceremonial. The real work happened in the twenty minutes before.

Campaigns that improve themselves

The next layer closes the loop.

The agent reads reply data daily. It finds the subject line winning a statistically meaningful test, kills the loser, writes a new challenger, and runs it again. Same for openers, CTAs, and pacing.

The loop runs itself between Tuesday and Thursday.

Cheap execution does not give you this. Scale plus memory does.
